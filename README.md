# 📊 Excel Data Analyzer Project

## 📌 Overview

This project focuses on analyzing structured data using Microsoft Excel. It demonstrates essential data handling techniques such as cleaning, transformation, and extracting meaningful insights for better decision-making.

---

## 🎯 Objective

* To understand and organize raw data
* To perform basic data analysis using Excel
* To generate useful insights from the dataset

---

## 📂 Project Files

* `Analyzer.xlsx` – Main Excel file containing dataset and analysis

---

## 🧹 Data Preparation

* Cleaned inconsistent and missing data
* Organized data into structured format
* Ensured accuracy and consistency

---

## ⚙️ Key Features

* Data cleaning and preprocessing
* Structured data organization
* Use of Excel formulas (SUM, AVERAGE, IF, etc.)
* Basic analytical approach
* Insight generation from raw data
